# search-Engine

<h1>Search Engine (Java & Python)</h1>
<p>This project is a desktop search engine application that combines a Java Swing Graphical User Interface (GUI) with a Python backend script powered by the Exa API for sophisticated web search and content summarization.</p>

<p>The architecture demonstrates effective cross-language communication and thread management to ensure a responsive user experience while executing external API calls.</p>
